Calc Dice  

ROLL!

Previous Rolls:

Key:

Quick Ref:

dice = d(6); 2d6 = d(6,2);

modify each roll: d(6,2,-1);

make variables: Lvl = 7;

use them: d(4,Lvl);

do math: d(6)*2+Lvl;

round down: rdown(d(6)/2);

round up: rup(2/5);

use any javascript functions: Math.Random();

===

Simple Examples:

Set a level:

L=6;

=


//Burning Hands

// 1d4 points of fire damage per caster level

d(4,L)

=


//Fireball

// deals 1d6 points of fire damage per caster level

d(6,L)

=

//Range: Long (400 ft. + 40 ft./level)

400+(40*L)

=


//Floating Disk

// Close (25 ft. + 5 ft./2 levels), It can hold 100 pounds of weight per caster level.

25+(5*(1+rdown(L / 2)))//Range

(100*L); //Can Hold

=


//Magic Missle

// d(dicetype,how many rolls, modifier per roll)

d(4,(1 + rdown(L / 2)), 1);

=
